Change log:

# --------------------------------------------------------------------------- #
Python 2.

Recommendation: Use Anaconda: Jupyter & Spyder.

Developed on Ubuntu 16.04 LTS and Windows 10 64-bit.
Documentation created in LaTeX with TeXstudio under Ubuntu and Windows
# --------------------------------------------------------------------------- #
Python libraries required:

Math
Numpy 
Scipy
Matplotlib including Basemap
Pandas
Datetime
Pytz
Aniso8601
SGP4 (Brandon Rhodes)

Note:
1. It is recommended to download these libraries using pip or conda install.
2. Basemap should be installed from source and not from conda install because
	the Anaconda version is barebones and cannot be used for plotting high
	resolution maps in this project.
