read_miscellaneous.txt

This directory contains miscellaneous scripts used for auxiliary work for the MeerKAT radar project

Author: Ashiv Rao Dhondea. ashivdhondea5[at]gmail.com
Created: 08 January 2018

# ------------------------------------------------------------------------------ #
Files in this directory: (Note that only python scripts are discussed here, the 
files created by these scripts are not discussed. They are named according to 
which script created them.

01. main_xxx_meerkat_layout.py
    Loads an .xlsx file containing MeerKAT dish latitudes and longitudes.
    Calculates the baselines between the dishes.
    Displays the location of the dishes on a Basemap map.
